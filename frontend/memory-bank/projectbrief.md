# Project Brief

## Project Name
Samsung Enterprise Web Application

## Core Requirements
- Enterprise-grade web application for Samsung internal use
- Built with React 18.2 and TypeScript (strict mode)
- Clean Architecture pattern for maintainability
- Multi-language support (Korean/English)
- High-performance UI with MUI and PrimeReact components

## Goals
- Maintain high code quality through strict coding standards
- Enable rapid feature development with consistent patterns
- Ensure security compliance for enterprise environment
- Support team scalability with clear conventions

## Scope

### In Scope
- Web application frontend (React SPA)
- State management with Redux Toolkit
- API integration with Axios
- Unit testing with Jest
- SCSS-based styling system

### Out of Scope
- Backend API development
- Mobile application
- Infrastructure/DevOps setup
